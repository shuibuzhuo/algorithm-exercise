# 深度优先遍历
尽可能深的搜索树的分支

类比：看书的时候

1. 看完书名
2. 看第一章
3. 看第一章的第1节、第2节...
4. 看第二章
5. 看第二章的第1节、第2节...

规律：

- 访问根节点
- 对根节点的 children 挨个进行深度优先遍历

# 广度优先遍历
- 先访问离根节点最近的节点

类比：看书的时候
1. 看完书名
2. 看第一章、第二章...
3. 再看第一章的第1节、第2节...
4. 看第二章的第1节、第2节...

规律

- 新建一个队列，把根节点入队
- 把队头出队，并访问
- 把对头的 children 挨个入队
- 重复第二、三步，直到队列为空
